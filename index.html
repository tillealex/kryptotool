Verfahrensinfo

Caesar-Verschlüsselung

Jeder Buchstabe wird im Alphabet um eine bestimmte Anzahl von Positionen verschoben.

Klartext: A B C D E F … X Y Z
Geheim: D E F G H I … A B C

Schnellhilfe

  1. Wähle links ein Verfahren.
  2. Gib deinen Text in der Mitte ein.
  3. Prüfe Schlüssel und Einstellungen.
  4. Lies unten das Ergebnis ab.
