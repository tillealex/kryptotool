# Dateivorlagen

In diesem Ordner werden die bisherigen einzelnen Kryptographie- und Geheimschrift-Tools gesammelt.

## Zweck

Die Dateien in diesem Ordner dienen als Ausgangsmaterial für die spätere Zusammenführung in die neue modulare Struktur des Kryptotools.

## Vorgehen

1. Bestehende HTML-, CSS- und JavaScript-Dateien hier hochladen.
2. Dateien möglichst unverändert lassen.
3. WILFRIED analysiert anschließend:
   - welche Funktionen übernommen werden,
   - welche Gestaltungselemente erhalten bleiben,
   - welche Logik in Module ausgelagert wird,
   - welche Teile neu geschrieben werden sollten.

## Wichtig

Dieser Ordner ist zunächst ein Sammelordner. Die eigentliche App-Struktur entsteht später in `css/`, `js/`, `assets/` und `index.html`.
